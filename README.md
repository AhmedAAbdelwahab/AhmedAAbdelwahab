## Featured Projects

### AI Diagnostic

AI-powered healthcare application combining:

- Brain Tumor Classification using ResNet50V2 with transfer learning and fine tuning.
- Parkinson Disease Detection using XGBoost trained on voice features.
- Audio preprocessing using Librosa, Parselmouth, and NoiseReduce.
- Flask REST APIs integrated with Flutter applications.
- Deployment using pyngrok and ngrok for remote inference.

### Concentrations AI

AI application for estimating KMnO4 solution concentrations from laboratory tube images.

- Built using a custom CNN model.
- Trained on a self-collected dataset of six concentration levels.
- Integrated with Flutter using Flask APIs and ngrok deployment.
